# H5中新语义化标签

```html
    <section> 定义文档中那个的主题部分的节、段 </section>

    <article>一个特数的section标签，定义一个独立的、完整的内容</article>

    <aside>用来装载页面中非征文的内容，独立与其他木块，例如广告、组成的连接、测表拦</aside>

    <header>定义文档、页面的页眉。通常是一些引导信息和导航栏</header>

    <footer>定义了文档、页面的也叫，和header类似</footer>

    <nav>定义了一个连接组组成的导航部分，其中的连接可以连接到其他网页或者当前页面的的其他地方</nav>

    <hgroup>用于对网页或者区段(section)的标题元素(h1~h6)进行组合</hgroup>

    <figure>用于对元素进行组合</figure>

    <figcaption>为figure元素添加天梯，一般放在figure第一个子元素或最后一个</figcaption>

    <details>定义元素的细节，用户可以点击查看或者隐藏</details>

    <summary>和details连用， 用来包含details的标题</summary>

    <canvas>用来进行canvas绘图</canvas>

    <video>定义视频</video>

    <audio>定义音频</audio>

    <embed>定义嵌入网页的内容，比如插件</embed>

    <source>该标签为媒体元素(比如video、audio)定义媒介元素<source>

    <datalist id='dl'>定义一可选数据的列表,与input配合(\<input list='dl'\>)制作下拉列表</datalist>

    <mark>展现突出文字</mark>

    <meter [min/max/low/high/optimum/value]></meter>

    <output></output>
```
